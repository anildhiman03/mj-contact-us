== Installation ==

install the plugin as we install the plugin. use short code [mjcontactus] to call in any page from admin section or call mj_contact_us() functin in any template file to call form directly


e.g.

1. Upload 'mj-contactus' file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[mjcontactus]` in your page editor or in templates paste this code mj_contact_us();
4. Do the admin setting as per your requirement
